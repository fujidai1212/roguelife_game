# フェーズ0 動作確認手順書

このドキュメントは、フェーズ0(プロジェクト土台)の成果物をスマホで動作確認するための手順です。

## 用意するもの

- お使いのスマホ(iPhone または Android)
- スマホに **Expo Go** アプリをインストールする(無料)
  - iPhone: App Store で「Expo Go」を検索してインストール
  - Android: Google Play で「Expo Go」を検索してインストール
- パソコンとスマホを**同じWi-Fi**に接続しておく(重要)

## 手順1: 開発サーバーを起動する

パソコンのターミナルで以下を実行します:

```bash
cd ~/game_dvp
npx expo start
```

しばらく待つと、ターミナルに**QRコード**が表示されます。

※ WSL(Windows上のLinux)環境ではスマホから接続できないことがあります。
その場合は QR コードの代わりにトンネル接続を使ってください:

```bash
npx expo start --tunnel
```

(初回は追加パッケージのインストールを聞かれるので y を入力)

## 手順2: スマホで開く

- **Android**: Expo Go アプリを開き、「Scan QR code」でターミナルのQRコードを読み取る
- **iPhone**: 標準のカメラアプリでQRコードを読み取り、表示されたリンクをタップ

## 手順3: 確認すること

以下がすべて確認できればフェーズ0は完了です:

1. **横画面(ランドスケープ)** で表示される(スマホの画面回転ロックは解除しておく)
2. **黒背景に白文字**の画面が表示される
3. 画面が横に3分割されている: 左=ボタン枠4つ / 中央=上からステータス帯・イラスト
   プレースホルダー・テキストログ / 右=ボタン枠4つ(今は空白)
4. 左のボタン4つ(辺りを見回す / 叫ぶ / もう一度眠る / ダイスを振る)がタップできる
5. ボタンをタップすると**ログが1行増え、自動で最下部までスクロールする**
6. 「ダイスを振る」を何度か押すと出目(1〜6)が変わる
7. 右の4スロットは空白の枠のまま(この場面では使わないボタン)

## 手順4: テストの確認(任意)

ゲームロジックのユニットテスト(自動テスト)が通ることも確認できます:

```bash
cd ~/game_dvp
npm test
```

「8 passed」と表示されれば成功です。

## 終了方法

ターミナルで `Ctrl + C` を押すと開発サーバーが止まります。
